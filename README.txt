###### lalana ######

#### IMPORTANT ####

This project requires some libraries that is available on the central maven repository.

Please run the `./install-libs.sh` on Linux or `./install-libs` on Windows.

To run the project, run the `./mvnw javafx:run` command.

===!> Don't forget to add a `.env` file at the project root 
	(there is a `.env.template` file at the project root that you can use as a reference).
